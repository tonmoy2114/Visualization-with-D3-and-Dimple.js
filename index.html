Visualization with D3 - Dimple.js

Visualization with D3 - Dimple.js

Overview

D3.js is a JavaScript library that helps to visualize data with the help of HTML, SVG, Canvas, and CSS. D3 allows the user to bind arbitrary data to a Document Object Model (DOM), and then apply data-driven transformations to the document. In this project, I have used D3.js along with dimple which is a library build on top of D3. Dimple utilize the power and flexibility of d3 and requires a gentle learning curve. Although basic programming knowledge is enough simple D3 projects, basic knowledge on the following languages will be helpful.

  • Basic HTML
  • Basic CSS
  • Basic JavaScript
  • SVG (optional)
  • Canvas (optional)

Resource

In order to use D3 and Dimple in any page, we must have to include d3.js and dimple.js in the page. Please download these two javascript files

  • D3.js
  • Dimple.js

You can also include this libraries directly from their server url. Like this

	<script src="http://d3js.org/d3.v3.min.js"></script>
<script src="http://dimplejs.org/dist/dimple.v2.0.0.min.js"></script>

Set up

We can usually view local HTML files directly from web browser. However, due to security reasons, the modern browser does not allow JavaScript to load local files. As a result, if D3 code needs to pull data from any external data files (like CSV or JSON), the browser will not allow that. That's why we will use a local server to render the visualizations. We can start a local server using Python. To start the local server using the python type this on the command line. 

	python -m SimpleHTTP

Now to browse the files, just type localhost:8000 into the address bar of the browser.

For Python 3 environment, we have to run this command instead 

	python -m http.server
